    Format a text

    In Google Docs, text can be formatted using menu icons or Format tab.

    Prerequisites:

    • Open a document in Google Docs and select the text.

    Format with icons

    To format a text, do the following:

    1. To change a font, select the font from the Font list.

    2. To change font size, select font size from the Font size list or enter required font size manually.

      • Optional: Click Decrease font size or Increase font size to gradually change font size.

    3. To change font style, click Bold, Italic, or Underline.

    4. To change text color, click Text color and select a color from a color picker.

    5. To highlight a text, click Highlight color and select a color from a color picker.

    Use format tab

    To format a text, do the following:

    1. To change font style, click Format > Text:
      • To make text bold, click Bold
      • To make text italic, click Italic
      • To make text underline, click Underline.
    2. To change font size, click Format > Text > Size:
      • To increase font size, click Increase font size.
      • To decrease font size click Decrease font size.
